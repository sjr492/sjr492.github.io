About

Whenever I get a new device or install a new game, the first thing I do is open the settings. Tuning performance mods and shaders in Minecraft, or finding the right balance between visual fidelity and frame rate, is genuinely more enjoyable to me than playing the game itself. Technologies like NVIDIA's DLSS fascinate me because they represent the kind of work I want to be close to: making things faster and smarter without compromise.

That same instinct is shaping how I approach my studies. I've been learning SQL on my own time because I find the challenge of querying and structuring data efficiently to be genuinely fun. The more I learn about data science, the more I want to understand how to use data not just correctly, but optimally. I want to be at the cutting edge of technical advancement in performance, whether that means systems, data, or the tools that connect them.
